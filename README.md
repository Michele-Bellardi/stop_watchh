# stop_watchh
## Descrizione del Progetto
Un'applicazione Flutter che riproduce il funzionamento di un classico cronometro.

## Struttura dell'applicazione
L'applicazione presenta una casella di testo al centro dello schermo che mostra i secondi e i minuti del cronometro.
In basso a destra si hanno 2 pulsanti che modificano lo stato dell'applicazione e cambiano in base al contesto:
  * il primo tasto può essere:
      * START: per far partire il cronometro,
      * STOP: per fermare il cronometro,
      * RESET: per ripristinare a 0 il cronometro;
  * il secondo tasto può essere:
      * PAUSE: per mettere in pausa il cronometro,
      * RESUME: per far ripartire il cronometro dopo la pausa.

## Per installare l'applicazione su Windows:
### Clonare la repository remota all'interno del computer:
da terminale, digitare: 
```bash
https://github.com/Michele-Bellardi/stop_watchh.git
```
### Avviare il progetto su Windows:
1. aprire il progetto con Android Studio o Visual Studio Code;
2. avviare un emulatore Android;
4. avviare il progetto.

## Per instalare l'applicazione su android:
Dopo aver aperto il progetto su Windows:
1. generare il file .apk da Android Studio;
2. trasferire il file generato all'interno della meoria del cellulare Android;
3. aprire il file .apk dal cellulare e seguire la procedura di installazione indicata;
4. aprire l'applicazione appena installata.
